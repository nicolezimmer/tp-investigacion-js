En el mercado existen las versiones de JavaScript 1.0 ,1.1 ,1.2 ,1.3 ,1.4 ,1.5 ,1.6 y 1.7. En 1997 se lanzó el estándar ECMAScript 1, al cual le siguieron las versiones 2, 3, 5, y 6.

¿Como se integra JavaScript a HTML?

Hay dos maneras de integrar js al archivo html:

Agregar JavaScript directamente al HTML:

Este metodo se hace mediante la etiqueta: "script" dentro de esta se trabaja todo el js. Se puede agregar tanto en el head como en el body.

Agregar JavaScript en un archivo separado:

Este es el metodo mas recomendable, ya que llegado un punto en el que el proyecyo se empieza a complejizar, hay varios html que necesitan conectarse con el mismo codigo js. Por esto es que se debe hacer todo el codigo de js en un archivo separado con la extencion ".js" luego en el html se le agrega la misma etiqueta que antes "script" y a esta se le agrega el atributo "src" para linkear el html al archivo js. esta etiqueta se puede agregar donde sea pero es mas recomendable agregarla al final del body.

¿Como es el manejo de las variables, tipos de datos y expresiones regulares en JS?

En js no hace falta declarar el tipo de datos que contienen las variables, ademas de esto tambien tenemos tres tipos de formas de declarar una variable; var, let y const. Aunque no hace falta declarar el tipo de dato de todas maneras existen dos grandes grupos:

Primitivos:

El tipo de variable mas básico. Si comparamos dos variables primitivas, tomamos el valor de dichas variables para esa comparación y no la referencia.

  • Undefined: variable a la que no se le asigno ningun valor.
  • Boolean: variable logica que puede tener dos valores: true y false
  • Number: uno de los dos tipos numericos con los que cuenta js, integra tambien los numeros de punto flotante. Incluye numeros entre -253 y 253
  • BigInt: tipo numerico para representar enteros muy grandes, llevan el sufijo “n”
  • String: representa datos textuales, cadena de caracteres
  • Symbol: representa un valor unico e inmutable
  • Null: representa solo un valor: null

Objetos:

toda variable que no es primitiva es considerada objeto. Al comparar variables objeto se toma la referencia (identificador que hace referencia a un lugar en la memoria) de dicha variables para la comparación y no el valor que puedan contener. Algunos tipos objetos son: Array, Set, WeakMap, WeakSet, Date y Function.

Expresiones regulares

Una expresion regular es una secuencia de caracteres que forman un patrón de busqueda, esto es útil para la búsqueda de patrones en cadenas de caracteres. Esto se crea con el constructor RegExp() y dentro de los paréntesis se agregan las banderas, que son las condiciones de búsqueda. Luego se puede aplicar la búsqueda con el metodo test; el cual devuelve un true si el patron se cumple o un false en caso contrario, o el método exec el cual devuelve un arreglo con todas las posiciones donde el patrón se cumplio.

¿Cuales son los operadores y conversion entre tipos?

Tenemos distintos tipos de operadores:

¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?

DOM, Document Object Model o modelo de objeto de documento, es la forma en que js puede acceder a un documento html, la cual es llevada con un documento en grupos de nodos y objetos. En otras palabras el DOM es el modelo que define de que manera se va a acceder a la parte estatica de una web con el objetivo de modificar la estructura, estilo y contenido del mismo y asi volverse dinsmica

¿Qué función tiene en JS el Object Map?

Un objeto map cumple una función muy similar a la de los objetos normales, la cual es almacenar pares clave-valor. Aunque cuenta con algunas diferencias. En los objetos normales las claves solo pueden ser de tipo string o symbol; en el caso de map pueden ser de cualquier tipo, con map podemos conocer su tamaño utilizando la propiedad size, y un map es iterable sin necesidad de las claves como si se necesitan con los objetos normales.

¿Cuáles son todos los eventos que trabaja JS?

Los eventos de js refieren a los tipos de acciones que pueden desencadenar un evento de nuestra eleccion

Hay muchos tipos de acciones, estos son la mayoria de eventos en js:

Relacionados con el mouse

  • onclick:Click sobre un elemento
  • ondblclick: Doble click sobre un elemento
  • onmousedown:Se pulsa un botón del ratón sobre un elemento
  • onmouseenter:El puntero del mouse entra en el area de un elemento
  • onmouseleave:El puntero del mouse sale del area de un elemento
  • onmousemove:El puntero del mouse se esta moviendo sobre el area de un elemento
  • onmouseover:El puntero del mouse se situa encima del area de un elemento
  • onmouseout:El puntero del mouse sale fuera del area del elemento o fuera de uno de sus hijos
  • onmouseup:Un boton del mouse se libera estando sobre un elemento
  • contextmenu:Se pulsa el boton derecho del mouse

Relacionados con el teclado

  • onkeydown: El usuario tiene pulsada una tecla (para elementos de formulario y body)
  • onkeypress: El usuario pulsa una tecla (momento justo en que la pulsa) (para elementos de formulario y body)
  • onkeyup: El usuario libera una tecla que tenía pulsada (para elementos de formulario y body)

Relacionados con formularios

  • onfocus: Un elemento del formulario toma el foco
  • onblur: Un elemento del formulario pierde el foco
  • onchange: Un elemento del formulario cambia
  • onselect: El usuario selecciona el texto de un elemento input o textarea
  • onsubmit: Se pulsa el boton de envio del formulario (antes del envio)
  • onreset: Se pulsa el boton reset del formulario

Relacionados con ventanas o frames

  • onload: Se ha completado la carga de la ventana
  • onunload: El usuario ha cerrado la ventana
  • onresize: El usuario ha cambiado el tamaño de la ventana

Relacionados con animaciones y transiciones

  • animationend, animationiteration, animationstart, beginEvent, endEvent, repeatEvent, transitionend

Relacionados con la bateria y carga de la bateria

  • chargingchange, chargingtimechange, dischargingtimechange, levelchange

Relacionados con llamadas tipo telefonia

  • alerting, busy, callschanged, connected, connecting, dialing, disconnected, disconnecting, error, held, holding, incoming, resuming, statechange

Relacionados con cambios en el DOM

  • DOMAttrModified, DOMCharacterDataModified, DOMContentLoaded, DOMElementNameChanged, DOMNodeInserted, DOMNodeInsertedIntoDocument, DOMNodeRemoved, DOMNodeRemovedFromDocument, DOMSubtreeModified

Relacionados con arrastre de elementos (drag and drop)

  • drag, dragend, dragenter, dragleave, dragover, dragstart, drop

Relacionados con video y audio

  • audioprocess, canplay, canplaythrough, durationchange, emptied, ended, ended, loadeddata, loadedmetadata, pause, play, playing, ratechange, seeked, seeking, stalled, suspend, timeupdate, volumechange, waiting, complete

Relacionados con la conexion a internet

  • disabled, enabled, offline, online, statuschange, connectionInfoUpdate
